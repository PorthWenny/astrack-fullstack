A campus navigation website specially designed for FAITH Colleges students. Astrack offers built-in facility reservation functionality and seamless map API integration to ensure effortless navigation around the campus.

## Features

- **Interactive Campus Map**: Navigate the FAITH Colleges campus with ease using the integrated map API.
- **Facility Reservation**: Users can reserve facilities directly through the platform.
- **User-Friendly and Responsive**: Provides a smooth and intuitive experience across devices.

## Technologies Used

Astrack Fullstack is proudly built utilizing MERN and modern web development technologies:

- **JavaScript** (69%): Implements site functionality and interactivity.
- **SCSS** (30.6%): Powers the design and styling with flexibility and scalability.
- **HTML** (0.4%): Provides the structure for the web pages.

## Getting Started

To deploy the project locally:

1. Clone the repository:

   ```sh
   git clone https://github.com/PorthWenny/astrack-fullstack.git
   ```
   
2. Navigate to the project folder:

   ```sh
   cd astrack-fullstack
   ```

3. Install dependencies:

   ```sh
   npm install
   ```

4. Start the development server:

   ```sh
   npm run start
   ```

5. Open your browser and navigate to `http://localhost:3000`.
